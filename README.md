This is a simple Python project that converts images into ASCII art
The program takes an image file, resizes it, converts it to grayscale and then transforms each pixel into ASCII characters to create a text version of the image

! How it works

1. Load an image using Pillow
2. Resize the image to make it suitable for ASCII conversion
3. Convert the image to grayscale
4. Map each pixel to a character from a list of ASCII characters
5. Print and save the final result into a text file

! Requirements

You need Python 3 and the Pillow library

! How to run

1. Place your image in the same folder as the script
2. Make sure the image name is "image.jpg"
3. Run the program
4. The result will be shown in the terminal and also saved in a file called "ascii_result.txt"

Sorry if it may not be optimized, but it works fine :)
